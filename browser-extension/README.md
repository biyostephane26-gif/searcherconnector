# Searcher Connector — Extension navigateur

## Installer (mode développeur, sans passer par le Chrome Web Store)

1. Ouvre `chrome://extensions` dans Chrome
2. Active "Mode développeur" (en haut à droite)
3. Clique "Charger l'extension non empaquetée"
4. Sélectionne ce dossier (`browser-extension/`)
5. Clique l'icône Searcher Connector dans la barre d'outils, colle ton token
   (généré dans l'app : Paramètres → Extension navigateur)

## Utilisation

Sur n'importe quelle page de candidature (LinkedIn, Upwork, site d'entreprise...),
un bouton doré apparaît en bas à droite : "🔍 Remplir avec Searcher Connector".
Un clic pré-remplit les champs reconnus (nom, email, téléphone, message) —
**l'envoi final reste toujours ton clic**, l'extension ne soumet jamais rien
elle-même.

## Publier sur le Chrome Web Store (plus tard)

Nécessite un compte développeur Google (frais unique ~5$) et une validation
Google (quelques jours). Pas nécessaire pour commencer à l'utiliser toi-même
ou en interne.
